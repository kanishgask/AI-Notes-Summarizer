# 🧠 AI Notes Summarizer Web App (React + Node + Python AI)

## 🚀 Overview
This project is a full-stack AI-powered Notes Summarizer that converts long text into short, meaningful summaries using NLP (Transformers model - BART).

It uses a **microservices architecture**:
- ⚛️ React → Frontend UI
- 🟢 Node.js → Backend API Gateway
- 🤖 Python (Flask) → AI Summarization Service

---

## 🏗️ Architecture
User → React Frontend → Node.js API → Python AI Service → Response → UI

---

## ✨ Features
- AI-powered text summarization
- Clean and responsive UI
- Microservices-based architecture
- REST API communication
- Scalable backend design
- Real-time response generation

---

### Backend
